---
description: 커밋 전 필수 수행 작업 (상태 업데이트 및 로그 기록)
---
1. `.context/CURRENT.md`를 방금 완료한 작업 상태, 현재 활성화된 Access URL, 다음 할 일로 상세 갱신
2. `.context/SESSIONS.md`에 오늘의 세션 로그 추가 (작업 날짜, 완료 내역, 발생한 이슈나 막힌 점)
3. 기존 기능이 망가지지 않았는지 프로젝트 정적 분석 또는 빌드 스크립트(예: `npm run build`) 실행
4. 터미널 결과가 성공적이면 커밋 진행. 커밋 메시지에 NOW(방금 한 것)/NEXT(다음 할 일)/BLOCK(막힌 점) 구조 포함
5. **최종 시각적 보고**: 프론트엔드 관련 작업인 경우, 브라우저를 통해 최종 화면을 확인하고 URL을 제공하거나 페이지를 열어 사용자에게 보여주며 세션 종료

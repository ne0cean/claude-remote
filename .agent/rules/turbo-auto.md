# ⚡ Turbo & Auto-Proceed Rules

에이전트가 사용자의 개입 없이 최대한 자율적으로 작업을 완수하기 위한 지침입니다.

## 1. SafeToAutoRun 활용
- 시스템 탐색(`ls`, `cat`), 상태 확인(`git status`), 테스트 실행 등 위험도가 낮은 명령은 `SafeToAutoRun: true`를 설정합니다.

## 2. 연속적 도구 호출
- 한 번의 사고 흐름에서 [분석 -> 수정 -> 검증] 과정을 묶어서 도구를 연속 호출하여 워크플로우를 가속화합니다.

## 3. 자율적 수정
- 도구 실행 실패 시 즉시 질문하지 않고, 원인을 스스로 파악하여 최소 2회 이상 재시도합니다.
